Class: UoACalendarClient

UoACalendarClient

new UoACalendarClient(apiToken, hostopt, portopt)

This class allows you to interact with the calendar backend for creating, retrieving and modifying calendar and event objects. To initialize the library you need to call the constructor, which takes as input a configuration object that can contain one or more of the following fields.
Parameters:
Name Type Attributes Description
apiToken string Sets the user's API token used for authentication purposes.
host string <optional>
 Authentication server to which the client will connect. Should *NOT* include the URL schema as it defaults to `http`. Defaults to `DEFAULT_HOST`.
port number <optional>
 TCP port from the host to which the client will connect. Defaults to `DEFAULT_PORT`.
